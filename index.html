Lists

Unorder List

Unorder lists are marked with bullet points and go in ul tags. Each list item goes in an li tag.

Order List

Orderes lists are numbered and go in ol tags. Each list item goes in an li tag.

Types of lists:

  • Unorder lists (ul)
  • Orderes lists (ol)

Anchors (i.e. links)

Anchor thags create links to other pages. The URL goes in the herf attribute. Go to DuckDuckGo to search for more information.

Setting target="_blank" will open the link in a new window. MDN for more details.

My HTML Cheat Sheet

Anchor tangs can also create links to a new location in the same page. Instead of putting a URL goes in the "herf" attribute, just use the "id" of the elemnt you want to navigate to. This allows you to Jump to the top of the page.

Text Emphasis

Italic

The em element adds extra emphasis to the text within the openeing and closing tabs. This shows uo as italic text in most browsers.

Bold

You can also use the strong element to emphasize text. This shows up as bold text in most browsers.
